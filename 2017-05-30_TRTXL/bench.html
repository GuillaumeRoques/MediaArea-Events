Performance (!!WIP!!)

From VIAA digitalized content (MXF/JP2K, SD 10-bit 4:2:2)
Preliminary results
With an Intel E5-2698V3 (16 cores, 2.3-3.6 GHz), FFmpeg 3.2 (without external libs)

single core:
- JP2k decoding: 0.16 fps / 2.3 MB/s
- JP2k decoding then FFV1 encoding: 0.14 fps / 2.0 MB/s (-10% files size)
- FFV1 checking by MediaConch: 0.55 fps / 7.5 MB/s
- FFV1 decoding: 0.50 fps / 6.5 MB/s
Scalability: for this CPU, x20
- FFV1 checking by MediaConch: 10 fps / 150 MB/s
Note: this is an average! lot of difference between files
